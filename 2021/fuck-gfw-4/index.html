想讲一下使用以来的感受，以及一些变化。

  • 目前使用了一年半，服务器没有中断过，出问题多半是客户端软件bug。目前使用的都是2dust大佬的v2rayNG和v2rayN，也在不断迭代中；
  • 之前用的是每月5刀的VPS，后来换成了目前最便宜的Vultr VPS，假装在是纽约，其实是新泽西，只有这里还有3.5刀的；
  • 域名首年8元，后面每年一百多。为了节约钱，我每年更换一次域名。只不过每次换的时候总会有意外，未来会写一个更换伪装域名的文档；
  • 萌精灵CDN现在改为使用Cloudflare的Global API登录，在CloudFlare - MyProfile - API tokens - Global API key；
  • 自选Cloudflare节点现在有更好的方法，使用XIU2大佬的测速脚本非常方便；
  • 在首次能上谷歌后，确认能通就行了，贸然去看油管测试网速，容易被封；
  • 233boy删除了github上所有的东西，除了自动部署脚本，该不会是被请喝茶了吧？但是v2ray这个关键词遍地都是，除了博客网站，甚至b站都有帖子。

优点

  • 每年$3.5\times6.5\times12\approx272$的服务器，加上每年8元的域名，每年不到300；
  • 速度快，前提是选择了合适的自选节点；
  • 不限设备；
  • 稳定；
  • 每月500GB流量(3.5刀)，我现在每月最多80GB；如果不够可以升级每月1T流量(升级为5刀服务器)；
  • 好玩；

缺点

  • 自己搭建自己维护，自己服务自己；
  • 每年需要更换域名(土豪请随意)；
  • 网速太快容易伤身/doge；
